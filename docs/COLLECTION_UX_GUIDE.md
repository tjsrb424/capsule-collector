# COLLECTION UX GUIDE

이 문서는 수집형 카드 컬렉션 욕구를 강화하는 UI/UX 원칙을 정리한다.  
원문 기준: `docs/CAPSULE_COLLECTOR_MASTER_DESIGN.md`

---

## 1) UX 핵심 감정(반드시 전달해야 하는 것)

- 캡슐을 열기 직전의 기대감
- 새 카드가 나왔을 때의 만족감
- 희귀 카드가 떴을 때의 짜릿함
- 도감 빈칸을 채워가는 수집 욕구
- 예쁜 카드를 바라보는 감성적 만족
- 중복이 나와도 성장에 쓰이는 안정감
- 오늘도 한 번 더 열고 싶은 가벼운 루틴감

---

## 2) 전체 UI 톤 & 원칙

- **모바일 세로 화면 최적화**
- **깔끔함 / 모던함 / 감성적**
- 과한 장식보다 **카드와 캡슐에 집중**
- 부드러운 그림자, 둥근 카드, 은은한 배경 그라데이션
- 보상 순간(캡슐 오픈/결과)에는 감정 강화를 위해 연출을 집중
- v0.1에서는 서버/로그인/결제/실제 광고 SDK를 붙이지 않고도 UX가 “완결”로 느껴지게 설계

---

## 3) 화면별 UX 구조

### 3.1 테마 홈(Theme Home)

우선순위:

1. 현재 선택 테마(이름/한 줄 설명)
2. 수집률(예: `12/80`)
3. 컬렉션 점수
4. 대표 카드(향후 기능, v0.1에서는 자리만 고려)
5. 캡슐 열기 CTA(가장 강한 버튼)
6. 광고 보너스 CTA(선택형)
7. 도감 진입

핵심 원칙:

- 복잡하지 않게, “한 번 더 열기” 동기가 바로 생기도록 구성

### 3.2 캡슐 오픈 화면(Open)

구성(권장):

- 선택한 캡슐(이름/설명)
- 캡슐 이미지(placeholder 가능)
- `1회 열기` (v0.1 핵심)
- `10회 열기` (v0.1에서는 비활성/비노출 가능)
- `광고 보고 열기` (mock)
- `확률 보기` (투명성/신뢰)

핵심 원칙:

- 조작은 단순하게, 기대감은 연출로 강화

### 3.3 결과 화면/결과 모달(Result)

신규 카드:

- `NEW!` 배지로 즉시 강화
- 카드명/등급/짧은 설명(또는 한 줄 문구)
- 다음 행동 CTA: `도감에서 보기`, `한 번 더 열기`

중복 카드:

- “중복 획득”이지만 실패로 느껴지지 않게 설계
- 중복 보상(커피 파우더 등)을 **즉시/크게** 보여주기
- 다음 행동 CTA: `한 번 더 열기`, (향후) `광고 보고 보상 2배`

희귀 이상:

- Rare 이상은 “기대감 딜레이 + 빛/글로우”로 감정 강화
- Legendary는 신비로운 잠금/등장(실루엣 → 공개) 등 단계감 부여

### 3.4 도감(컬렉션) 화면(Collection)

도감은 단순 목록이 아니라, 다음 욕구를 만들어야 한다.

- 수집 진행도
- 미획득 카드의 궁금증
- 카테고리 완성 목표
- 희귀 카드 보유 자랑
- 카드 상세 감상
- 대표 카드 설정 욕구(향후)
- 다시 캡슐을 열고 싶은 동기

권장 구성:

- 전체 수집률(예: `12/80`)
- 컬렉션 점수
- 카테고리 리스트(각 카테고리 `x/8` 진행도)
- 카드 그리드: 획득/미획득이 한눈에 구분

### 3.5 카드 상세 모달(Card Detail)

상세 모달 정보(기획 기준):

- 큰 카드 이미지(또는 placeholder)
- 카드명/등급/카테고리
- 설명
- 획득일(향후)
- 보유 수량/별 등급(향후 성장)
- 다음 별까지 필요한 중복 수량(향후)
- 컬렉션 점수(또는 기여 점수)
- 대표 카드로 설정 버튼(향후)

## 4) 카드 상태별 “느낌” 차별화 가이드

### 4.1 신규 카드(New)

- “처음 얻었다”가 0.2초 안에 이해되어야 함
- NEW 배지, 밝은 하이라이트, 작은 축하 연출(과하지 않게)

### 4.2 중복 카드(Duplicate)

- “아쉽다”보다 “좋다(재화가 쌓인다)”가 먼저 와야 함
- 중복 보상은 숫자로 크게, 색으로 따뜻하게(커피 톤)

### 4.3 희귀 카드(Rare / Signature)

- 공개 전 딜레이로 기대감 만들기
- 배경 톤/글로우/사운드(향후)로 등급 체감 강화

### 4.4 전설 카드(Legendary)

- 가장 강한 희소성 + 가장 큰 보상감
- 실루엣/잠금 느낌 → 빛 → 공개(단계감)
- 과한 애니메이션은 피하고, “정적 + 강한 한 방” 느낌

---

## 5) 수집 욕구 강화 장치(기획 기준)

- **미획득 카드 표현**
  - 어두운 실루엣
  - 등급만 표시 가능
  - 이름은 `????` 또는 잠김 처리
  - 카테고리 위치는 유지
  - Legendary는 더 신비로운 잠금 연출

- **수집률/진행도**
  - 전체 수집률과 카테고리별 진행도는 항상 노출(동기 유지)

- **카테고리 완성 목표**
  - `x/8` 형태로 “끝까지 채우고 싶게” 만들기
  - v0.1에서는 보상 UI만 기획해두고, 실제 지급은 v0.2+

- **대표 카드**
  - 유저가 자랑하고 싶게 만드는 장치(향후 구현)

- **최근 획득 카드(히스토리)**
  - 성취를 되새기게 하고, 중복이어도 누적감을 주는 장치(향후 구현)

- [ ] 도감에서 획득/미획득이 한눈에 구분됨
