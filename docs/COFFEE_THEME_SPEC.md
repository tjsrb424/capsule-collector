# COFFEE THEME SPEC

이 문서는 `캡슐 컬렉터`의 첫 구현 테마인 **커피(coffee)** 테마 상세 기획을 정리한다.  
원문 기준: `docs/CAPSULE_COLLECTOR_MASTER_DESIGN.md`

---

## 1) 테마 개요

- **themeId**: `coffee`
- **컨셉**: Coffee 2048에 등장하는 음료 **80종**을 카드화한 수집 테마
- **v0.1 목표**: 커피 테마 하나로 캡슐 컬렉터의 기본 재미(캡슐 오픈/도감/점수/저장)를 검증

---

## 2) 카드 구성

### 2.1 카드 수

- 총 **80장**

### 2.2 등급 구조

- `common`
- `uncommon`
- `rare`
- `signature`
- `legendary`

### 2.3 카테고리 구조(10개, 각 8장)

```text
1. 에스프레소/기본 커피
2. 기본 우유 커피
3. 시럽/달달 라떼
4. 모카/디저트 커피
5. 말차/티 라떼
6. 청량/아이스/논커피
7. 희귀 재료 음료
8. 시간대 한정 음료
9. 시그니처 음료
10. 전설/고급 컬렉션 음료
```

---

## 3) 카테고리 역할 요약

### 3.1 에스프레소/기본 커피

- 초반 “첫 수집 성취”를 제공하는 입문 카드군
- `common/uncommon` 중심으로 빠르게 도감이 채워져야 함
- 대표 카드 예시: 아메리카노, 에스프레소, 도피오, 룽고, 리스트레토, 카페 크레마, 콜드브루, 롱 블랙

### 3.2 기본 우유 커피

- 부드러운 비주얼로 “예쁜 카드” 만족도를 올리는 구간
- Rare 수집 욕구의 첫 목표 제공

### 3.3 시럽/달달 라떼

- 색감/토핑 차별화가 명확한 수집군
- Rare 수집 욕구 강화

### 3.4 모카/디저트 커피

- 디저트성 비주얼로 고급감 강화
- Signature 기대감의 시작점

### 3.5 말차/티 라떼

- 커피 톤에서 벗어난 색상(초록/차색 등)으로 시각적 변주 제공

### 3.6 청량/아이스/논커피

- 얼음/탄산/과일/투명감으로 신선함 제공
- 여름 이벤트/광고 보상 캡슐과 궁합이 좋음

### 3.7 희귀 재료 음료

- 프리미엄 재료감으로 Signature 밀도를 높이는 상위 구간

### 3.8 시간대 한정 음료

- 시간 조건으로 “접속 이유”를 제공하는 이벤트성 카드군
- 광고 보고 추가 열기와 궁합이 좋음

### 3.9 시그니처 음료

- 테마의 정체성을 강화하는 고급 대표 카드군
- 고급 연출/대표 카드 욕구 강화 대상

### 3.10 전설/고급 컬렉션 음료

- 장기 플레이 최종 목표 카드군
- Legendary 연출/희소성의 핵심 대상

## 4) 캡슐 설계

### 4.1 v0.1 캡슐

#### 기본 커피 캡슐

- 모든 커피 카드가 등장하는 기본 캡슐
- 드랍 확률:
  - Common 58%
  - Uncommon 28%
  - Rare 12%
  - Signature 1.8%
  - Legendary 0.2%

#### 광고 보너스 커피 캡슐

- 광고 시청 후 열 수 있는 보너스 캡슐(선택형 보상)
- 드랍 확률:
  - Common 45%
  - Uncommon 32%
  - Rare 18%
  - Signature 4.5%
  - Legendary 0.5%
- 일일 제한 예시: `dailyAdLimit: 10`

### 4.2 v0.2+ 이후 캡슐(확장안)

- 라떼 캡슐: 기본 우유 커피/시럽 라떼 중심
- 디저트 캡슐: 모카/디저트 커피 중심
- 티 라떼 캡슐: 말차/티 라떼 중심
- 청량 캡슐: 청량/아이스/논커피 중심
- 시간대 캡슐: 현재 시간대 한정 음료 확률 상승
- 시그니처 캡슐: Rare 이상 중심
- 전설 캡슐: Rare/Signature/Legendary만 등장

## 7) 80장 데이터 입력 시 필드 규칙(권장)

> 구현 타입과 1:1로 고정하기보다는, 데이터 작성 시 “일관성”을 위한 규칙을 우선한다.

### 7.1 공통 필드

- `id`: 영문 소문자 + `_` 권장(예: `cafe_latte`, `cold_brew`)
- `themeId`: 항상 `coffee`
- `nameKo`: 한글 카드명
- `rarity`: 등급(`common|uncommon|rare|signature|legendary`)
- `categoryId`: 10개 카테고리 중 1개를 참조

### 7.2 선택 필드(확장 대비)

- `description`: 카드 설명(향후 상세 모달에 사용)
- `sourceGame`: `Coffee 2048` 등 출처 표기(향후 크로스 프로모션)
- `imagePath`: 카드 이미지 경로(없으면 placeholder)
- `silhouetteImagePath`: 미획득 실루엣 경로(없으면 공통 실루엣)
- `timeLimited`: `"morning" | "day" | "evening" | "night"` (시간대 한정 카테고리에서 사용 가능)
